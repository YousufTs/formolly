# 💕 Наши 3 дня вдвоём

Романтический сайт-анкета для любимой: она отвечает на тёплые вопросы про идеальное трёхдневное свидание, еду, близкие и бонусные пожелания, а в конце одним нажатием отправляет тебе все ответы. Сайт адаптирован под телефон.

## ✨ Что внутри
- Пошаговая анкета с красивым мобильным дизайном и плавающими сердечками
- Разделы: **3 дня свидания** · **что покушать** · **близкие пожелания** · **бонусные вопросы от тебя**
- Поля: текст, варианты-чипсы, слайдер настроения
- Прогресс сохраняется в браузере (можно закрыть и вернуться)
- Кнопки отправки результата в **Telegram**, **WhatsApp** или **копирование** текста

## 🛠 Как настроить под себя
Открой файл `script.js` и поправь блок `CONFIG` в самом верху:

```js
const CONFIG = {
  herName: "Жанна",          // имя твоей девушки

  // Автоматическая отправка её ответов тебе (заполни ОДИН вариант):
  telegramBotToken: "",      // токен бота от @BotFather — ответы прилетят тебе в Telegram
  telegramChatId: "",        // твой chat id (узнать у @userinfobot)
  formspreeId: "",           // ИЛИ id формы Formspree — ответы придут на e-mail

  music: true,               // романтическая фоновая музыка (с кнопкой вкл/выкл)
};
```

### 📩 Чтобы её ответы приходили тебе автоматически
**Вариант A — Telegram-бот (рекомендую, мгновенно):**
1. В Telegram открой **@BotFather** → команда `/newbot` → придумай имя → получишь **токен**.
2. Напиши своему новому боту любое сообщение (чтобы он мог тебе писать).
3. Узнай свой **chat id** у бота **@userinfobot**.
4. Вставь оба значения в `telegramBotToken` и `telegramChatId`.

**Вариант B — Formspree (на e-mail):**
1. Зарегистрируйся на [formspree.io](https://formspree.io), создай форму.
2. Скопируй id из ссылки вида `formspree.io/f/abcwxyz` → вставь в `formspreeId`.

Когда Жанна нажмёт «Отправить мои ответы», они уйдут тебе сами. Если ничего не настроишь — остаются запасные кнопки «поделиться в Telegram/WhatsApp» и «скопировать».

### 🎵 Музыка
`music: true` включает нежную мелодию, которая генерируется прямо в браузере (без файлов). В правом верхнем углу есть кнопка 🎵 для вкл/выкл.

Сами вопросы можно менять в массиве `SECTIONS` ниже — добавляй, убирай и переписывай свободно.

## 🚀 Публикация на GitHub Pages
1. Влей эту ветку в `main` (через Pull Request).
2. Зайди в **Settings → Pages** репозитория.
3. В разделе **Build and deployment** выбери источник **GitHub Actions**.
4. Через минуту сайт будет доступен по адресу:
   `https://<твой-логин>.github.io/formolly/`

> В репозитории уже лежит автодеплой `.github/workflows/pages.yml`, который публикует сайт при каждом пуше в `main`.
